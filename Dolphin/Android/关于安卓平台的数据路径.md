# Android的默认路径

根据 [官方wiki](https://wiki.dolphin-emu.org/index.php?title=Controlling_the_Global_User_Directory#Android)(截至20260330) 的信息，目前安卓版本的路径共分为两个：

- 旧版： `/storage/emulated/0/dolphin-emu`
- 新版： `/storage/emulated/0/Android/data/org.dolphinemu.dolphinemu/files`

造成这件事的根本原因是傻*谷歌要求、并且开发组要上play store，所以新版只能画蛇添足似的加了个数据导入导出（比起这个还不如学ppsspp、eden和azahar那样默认在谷歌空间里但允许用户自己指定数据目录位置），但这并不方便使用类似Syncthing之类的软件同步数据，并且还会在卸载模拟器的时候不小心删除数据。

# 模拟器允许的其他方案

不过在阅读官方wiki的时候发现了一件事，也是这篇经验的由来，即：
> However, if all of the following applies, the old dolphin-emu location will be used as the user directory instead
> 然而，如果同时满足以下所有条件，则会改用旧的 dolphin-emu 位置作为用户目录

这段里，列出的要求是：
- `Scoped storage` 被禁用；
- `Android/data/org.dolphinemu.dolphinemu/files` 目录不存在或为空；
- `dolphin-emu` 目录存在；
- 当 Dolphin 请求访问您的文件权限时，允许权限。

后面又提到，不能触发 preserveLegacyExternalStorage 错误，但这个错误是仅限安卓11，并且dolphin版本为 5.0-15348 或更新版本，所以这个错误一般情况下遇不到（除非你用老款安卓掌机）。

# 在获取了权限的情况下使用新版

然后我想到，如果先装旧版，在旧版数据存在的情况下更新到最新版会怎样？遂进行实测，得到了：

> 最新版本的情况下，还可以使用 dolphin-emu 目录储存模拟器数据

这个可以说是最理想的结果。换句话说，现在可以在享受最新版的各种特性（比如支持gpu驱动安装了）的基础上，还享受旧版本的非强制目录的好处。

这时候就有人说了，如果我直接在根目录新建 `dolphin-emu` 目录呢？直接把旧数据保留但不装旧apk只装新apk呢？很遗憾，并不行，这里应该是要借助旧版获取权限，然后新版通过权限进行条件判断，根目录的条件的优先级低于data目录，所以只有数据是不行的，因为没有相应的权限，在获取权限之前就已经做完了路径判断，所以也就无法直接只安装新版来实现外置目录。

不过，我只进行了很基础的测试，并在最新版的User Data选项内查看数据目录的地址（这个选项的子菜单里会显示模拟器的数据目录位置），并且这个方案也会受到诸如 `安卓版本更新导致无法安装5.0-15341版本`、`官方新版的包名变动导致无法覆盖安装`、`Dolphin官方版本模拟器更新导致数据目录规则变更` 等额外因素使之无效，所以这里仅以临时解决方案的形式存在，并在最近的将来保证其可用（不过安卓版至少15341就已经升级到了64位，理论上是只要系统版本满足需求就能装了）。

# 使用根目录下的目录作为数据储存的具体方法

那么最后总结一下，如果希望获得新版Dolphin的各种特性，但又想使用机器内部储存根目录的 `dolphin-emu` 目录储存模拟器数据，则按照以下步骤进行是可以实现的：

1. 在 [这里](https://dl.dolphin-emu.org/builds/e6/5b/dolphin-master-5.0-15341.apk) (如果官方地址失效了来[这里](https://web.archive.org/web/20251209221828/https://dl.dolphin-emu.org/builds/e6/5b/dolphin-master-5.0-15341.apk)，需要梯子) 下载 `dolphin-master-5.0-15341.apk` ；
2. 安装 `dolphin-master-5.0-15341.apk` ；
3. 点开模拟器的图标，这时候会请求权限，允许权限，然后退出或者后台模拟器；
4. 来 [这里](https://cn.dolphin-emu.org/download/) 下载官方最新的发行版，截至20260330，目前最新的版本是2603a。
5. 安装刚下好的最新版的apk；
6. 安装完毕打开模拟器，然后无论你是横屏还是竖屏，均点击上方的齿轮图标，打开设置页面；
7. 点击 `Config` → `User Data` 到达用户数据界面；
8. 此时如果界面提示两行英文（大意是用户数据储存在卸载的时候不会被删除的位置），并且在 `Open User Data Folder` 按钮上方的地址显示为 `/storage/emulated/0/dolphin-emu` 则大功告成！
