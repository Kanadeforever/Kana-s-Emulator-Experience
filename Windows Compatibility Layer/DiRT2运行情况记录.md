# 运行情况综述

总体来说，只要设置对了兼容层，那么基本上都可以跑，但摸索的过程确实折磨人。
所以这里记录能运行的情况，包括一些奇形怪状的过程。

# GameHub Lite的情况
重点设置注意：
  - 需要x86_64的wine/proton
  - Box64选择0.38/0.40都可以
  - dxvk选择最稳定的1.10.3-async
  - Box64的运行档位选择高性能，并把SafeFlags设置成2
  - 组件依赖openal必装，没有就用openal安装程序
  - 第一次进游戏可能需要进容器里打开位于文档的config.xml取消全屏显示

补充一部分reddit上的信息：
  - 兼容层：wine 9.5-x64-1
  - 翻译参数：通用
  - GPU 驱动：8Elite-800.51
  - DXVK：dxvk-2.6-arm64ec-async
  - CPU 翻译：Box64-0.37

以上最多只能到标题界面，进游戏就闪退

# GameNative的情况

重点设置，其余默认：
  - wine版本选择proton-9.0-x86_64
  - 先去全局设置里找到BOX64预设集，然后复制一份“高性能”设置（我命名为高性能【安全】），再把 `BOX64_DYNAREC_SAFEFLAGS`的值设置为2，并保存
  - 上面完成后游戏设置里，BOX64版本选择0.4.0，然后预设集选刚配置的“高性能【安全】”；
  - 其余几乎没有需要改的地方
  - 唯一需要注意的是，可能需要加一个openal32.dll给游戏，非必须，未验证
